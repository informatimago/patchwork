Comment rapporter une bogue - Bug Tracking: How to add a Ticket

N'importe qui peut ajouter un rapport de bogue, mais tous doivent s'identifier.

Vous pouvez créer un compte, ou vous connecter avec un compte existant.

Anybody can add a bug report, but everybody must log in.

You may create an account, or log in with an existing account.

Comment clôner le dépôt - How to clone the repository

Le projet utilise le logiciel de gestion des sources git.

À l'origine, le projet utilisait le logiciel de gestion des sources fossil. Fossil est encore utilisé, pour le suivi des bogues, et pour ce wiki, mais les sources sont gérées par git, avec un dépôt central sur gitorious: https://gitorious.org/patchwork
